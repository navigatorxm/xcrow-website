> Digital Escrow Payment Protection Platform

This repository contains a React + Vite + TypeScript clone of the Xcrow website.

## Tech Stack

- **Frontend**: React 18 + TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Package Manager**: npm

## Features

- Responsive design for desktop, tablet, and mobile
- Smooth animations and transitions
- Feature carousel component
- Blog insights section with article cards
- Newsletter signup form
- Comprehensive footer with multiple navigation sections

## Project Structure

```
xcrow-website/
├── src/
│   ├── components/        # Reusable components
│   ├── sections/          # Page sections
│   ├── App.tsx           # Main app component
│   └── index.tsx         # Entry point

```

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The app will be available at `http://localhost:5173`

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
